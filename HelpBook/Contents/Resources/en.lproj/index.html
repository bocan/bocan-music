Bòcan Music

Native macOS music player

Getting Started

Add music to your library
Choose File → Add Folder to Library… (⌘⇧O) or File → Add Files to Library… to point Bòcan at your music. The library scanner will index audio files and read their tags automatically.
Playing tracks
Double-click any track to start playback. Use Space to play/pause, ⌘→ for next, and ⌘← for previous.
Up Next queue
Right-click tracks and choose Add to Queue, or drag them onto the Up Next sidebar section. The current queue is shown under Playback → Show Up Next (⌘⌥U).
Editing track info
Select one or more tracks and press ⌘I, or choose Track → Get Info. The editor lets you update tags, artwork, and lyrics for a single track or in bulk.
Playlists
Create standard playlists with File → New Playlist… (⌘N) or rules-based Smart Playlists with File → New Smart Playlist… (⌘⌥N). Import existing M3U/PLS/XSPF playlists via File → Import Playlist….
Lyrics
Toggle the lyrics pane with ⌘⌥L. Bòcan displays embedded LRC timestamps when available and scrolls in sync with playback.
Miniplayer
Switch to the compact window with ⌘⌥M or Window → Toggle Miniplayer.
Scrobbling
Connect your Last.fm account under Bòcan → Settings… → Scrobbling to enable automatic track scrobbling.

Supported Formats

Bòcan plays all formats supported by macOS Core Audio plus additional formats via its built-in FFmpeg engine:

  • Core Audio: FLAC, ALAC/M4A, MP3, AAC, AIFF, WAV, CAF
  • FFmpeg engine: OGG Vorbis, Opus, APE (Monkey's Audio), DSD (DSF/DFF)

Tag formats supported: ID3v2 (MP3), Vorbis Comments (FLAC/OGG/Opus), MP4/iTunes tags (M4A/AAC), APEv2.

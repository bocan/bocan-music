# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| Latest | ✅ |
| Older  | ❌ |

## Reporting a vulnerability

**Please do not file a public GitHub issue for security vulnerabilities.**

Report security issues privately via GitHub's [Security Advisory](https://github.com/bocan/bocan-music/security/advisories/new) feature, or email **chris@funderburg.me**.

Include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested mitigations

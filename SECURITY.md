# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| Latest | ✅ |
| Older  | ❌ |

## Reporting a vulnerability

**Please do not file a public GitHub issue for security vulnerabilities.**

Report security issues privately via GitHub's [Security Advisory](https://github.com/cloudcauldron/bocan-music/security/advisories/new) feature, or email **security@cloudcauldron.io**.

Include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested mitigations

We aim to acknowledge reports within 48 hours and provide a fix or mitigation plan within 14 days.
